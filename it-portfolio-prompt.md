# 🚀 Prompt Tạo IT Portfolio

## Prompt Tiếng Việt

> Tạo một trang web portfolio cá nhân cho tôi — một lập trình viên **[Frontend / Backend / Fullstack / DevOps / AI Engineer]** với **[X năm]** kinh nghiệm.
>
> **Thông tin cá nhân:**
> - Tên: **[Tên của bạn]**
> - Vị trí: **[Ví dụ: Senior Frontend Engineer]**
> - Tagline: **[Ví dụ: "Tôi xây dựng những thứ chạy được — và trông đẹp."]**
> - Email: **[email@example.com]**
> - GitHub: **[github.com/username]**
> - LinkedIn: **[linkedin.com/in/username]**
>
> **Kỹ năng:** React, TypeScript, Node.js, Docker, PostgreSQL, AWS *(thay bằng stack của bạn)*
>
> **Dự án nổi bật (liệt kê 3):**
> 1. **Tên dự án** — Mô tả ngắn, tech stack, link demo/GitHub
> 2. **Tên dự án** — Mô tả ngắn, tech stack, link demo/GitHub
> 3. **Tên dự án** — Mô tả ngắn, tech stack, link demo/GitHub
>
> **Yêu cầu thiết kế:**
> - Giao diện dark theme, phong cách terminal/hacker hoặc minimalist hiện đại
> - Animation mượt khi scroll và hover
> - Responsive trên mobile
> - Phần: Hero → About → Skills → Projects → Contact
> - Code trong một file HTML duy nhất, không cần framework ngoài
>
> Hãy tạo trang portfolio hoàn chỉnh, production-ready, độc đáo — **không dùng template nhàm chán**.

---

## Prompt English

> Create a stunning, production-ready personal portfolio website for me — a **[your role]** developer with **[X years]** of experience.
>
> **Personal info:** Name: [Name] | Role: [Role] | Email: [email] | GitHub: [url] | LinkedIn: [url]
>
> **Tech stack:** [your skills]
>
> **3 featured projects:** [project name, description, tech, link] × 3
>
> **Design direction:** Choose a bold, memorable aesthetic — dark terminal vibe, brutalist grid, or refined monochrome. Include: smooth scroll animations, hover micro-interactions, mobile-responsive layout, sections: Hero / About / Skills / Projects / Contact. Deliver as a single self-contained HTML file. Make it unforgettable.

---

## 💡 Tips Tuỳ Chỉnh

| Muốn gì | Thêm vào prompt |
|---|---|
| Dark hacker style | `"dark terminal aesthetic, monospace font, green-on-black"` |
| Tối giản sang trọng | `"minimal, lots of whitespace, serif typography, luxury feel"` |
| Màu sắc cụ thể | `"color palette: navy + gold"` |
| Có ảnh avatar | `"include a placeholder for profile photo"` |
| Có nút tải CV | `"add a Download CV button"` |
| Hiệu ứng 3D | `"add subtle 3D tilt effect on project cards"` |
| Đa ngôn ngữ | `"support both Vietnamese and English with a toggle button"` |

---

## 📐 Cấu Trúc Sections Gợi Ý

```
Hero        → Tên, role, tagline, CTA button, avatar
About       → Giới thiệu bản thân, kinh nghiệm, passion
Skills      → Tech stack dạng badge/icon/progress bar
Projects    → Card từng dự án: mô tả, tech, link demo & GitHub
Experience  → Timeline công việc (tuỳ chọn)
Contact     → Form liên hệ hoặc link mạng xã hội
```

## 🎨 Phong Cách Thiết Kế Phổ Biến

### Terminal / Hacker
```
"dark background #0d0d0d, monospace font (JetBrains Mono),
neon green accent #00ff41, typing animation on hero text,
ASCII art decorations, blinking cursor effect"
```

### Minimalist Hiện Đại
```
"off-white background, generous whitespace, serif display font,
single accent color, clean grid layout, subtle hover transitions"
```

### Brutalist
```
"raw grid, bold typography, high contrast black/white,
thick borders, no rounded corners, bold typographic hierarchy"
```

### Glassmorphism
```
"dark gradient background, frosted glass cards,
blur effects, subtle glow, Inter or Outfit font"
```
